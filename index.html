怎么看图 · 找训练方向

① 先看懂各个显示

  • 谐波 H1–H5 + 频谱倾斜：谐波结构与音色明暗。倾斜越平缓 = 高频越多 = 越明亮。
  • 二维特征垫：横轴 左暗→右亮；纵轴 上「已按压」→下「更轻柔(气声)」。看那个点落在哪。
  • 基频轨迹：音高(F0)随时间。开启目标线把目标音高钉住对照。
  • F1 / F2 共振峰：声道共鸣——决定听感性别比音高更关键。整体越高越偏女、越低越偏男。典型参考：男声约 F1 480–650 / F2 1100–1500 Hz，女声约 F1 600–800 / F2 1700–2300 Hz（Hillenbrand 1995）。

② 女性化方向（MtF）

  • 音高 F0：抬到约 165–220 Hz，先在目标线上稳住。
  • 共鸣 F1/F2：抬高——舌前移、唇展(微笑)、喉位略上抬、用口腔前部；F2 升高最关键。目标参考：F2 抬向 ~1700–2300 Hz、F1 ~600–800 Hz（女声典型）。
  • 明暗：偏亮（频谱倾斜更平，点往右）。
  • 气声：略带气声有帮助（H1–H2 略大，点略偏下）。

③ 男性化方向（FtM）

  • 音高 F0：降到约 100–135 Hz。
  • 共鸣 F1/F2：降低——喉位放松下沉、扩大口腔后部空间、声道拉长；F2 降低。目标参考：F2 降向 ~1100–1500 Hz、F1 ~450–650 Hz（男声典型）。
  • 明暗：偏暗（频谱倾斜更陡，点往左）。
  • 气声：更结实/压声（H1–H2 小，点偏上）。

④ 怎么找方向、怎么练

  • 对照录音报告里你的 F1/F2 中位 与上面卡片的参考区间，差多少就往哪个方向调。
  • 一次只练一个维度：先把音高稳定在目标线，再调共鸣(F2)，最后修明暗/气声。
  • 共鸣比音高更影响听感性别——别只盯着 F0，多观察 F2 的升降。
  • 参考值发持续元音时更接近，连续说话整体偏低——重点看变化趋势与共鸣图位置；目标因人而异，长期训练建议配合言语治疗师。
